# Arşiv (apps/\_arsiv)

Terk edilen uygulamalar buraya taşınır (süreç: docs/SURECLER.md §4).

- Bu klasör pnpm workspace ve `pnpm verify` KAPSAMI DIŞINDADIR:
  çekirdek değişiklikleri arşivdeki uygulamaları derlemez.
- Arşivlemek, uygulamayı mağazadan kaldırmaz; o ayrı bir Seviye 3 karardır.
- Geri getirme: klasörü `apps/` altına taşı → `pnpm install` → `pnpm verify`.
